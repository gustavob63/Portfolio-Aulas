Ementa

Disciplina sobre fundamentos de Interação Humano-Computador (IHC), com foco em UX, UI e design de interação. Ensina desenvolvimento web com HTML, CSS e JavaScript, incluindo bibliotecas e frameworks para criar interfaces responsivas, acessíveis e interativas, unindo teoria e prática na construção de soluções digitais funcionais e atrativas.

Habilidades

Analisar e interpretar fundamentos da Interação Humano-Computador (IHC) e sua importância para sistemas digitais centrados no usuário.

Avaliar criticamente a usabilidade de interfaces com base em heurísticas, acessibilidade e design centrado no usuário.

Projetar interfaces digitais responsivas e acessíveis , aplicando conceitos de UX/UI em ambientes web.

Codificar páginas web estruturadas com HTML , organizando cabeçalhos, parágrafos, listas, formulários e links.

Aplicar regras de estilo com CSS , criando layouts com cores, tipografias, grids e posicionamento de elementos.

Criar interatividade com JavaScript, manipulando eventos, elementos do DOM, variáveis e estruturas condicionais.

Utilizar frameworks e bibliotecas modernas(como Bootstrap e jQuery) para agilizar o desenvolvimento e garantir responsividade e padronização visual.

Construir protótipos navegáveis e iterativos, usando ferramentas digitais de design e testes com usuários.

Colaborar em projetos em equipe, organizando tarefas, comunicando ideias e utilizando controle de versão (como Git).

Aplicar princípios de acessibilidade digital, assegurando que os produtos sejam utilizáveis por pessoas com diferentes necessidades.

Competências

1. Compreender IHC e aplicar no desenvolvimento de interfaces centradas no usuário.

2. Aplicar princípios de UX/UI no design e avaliação de aplicações digitais.

3. Estruturar páginas web com HTML, seguindo padrões e boas práticas.

4. Criar layouts atrativos, responsivos e acessíveis com CSS.

5. Adicionar interatividade com JavaScript básico, manipulando DOM, eventos e lógica.

6. Usar frameworks e bibliotecas modernas (Bootstrap, jQuery etc.) para agilizar o desenvolvimento.

7. Trabalhar em equipe e desenvolver projetos colaborativos, seguindo metodologias ágeis e prototipação iterativa.

8. Analisar criticamente soluções digitais considerando acessibilidade, usabilidade e experiência do usuário.

Conteúdo Programático

Unidade 1

    Interação Humano-Computador (IHC): conceitos, histórico, princípios, ergonomia, normas e padrões de usabilidade.

Unidade 2

    User Experience (UX) e User Interface (UI): fundamentos, heurísticas de Nielsen, design centrado no usuário, arquitetura da informação, acessibilidade e responsividade.

Unidade 3

    HTML (Hypertext Markup Language): estrutura básica, semântica, tags estruturais e formulários acessíveis.

Unidade 4

    CSS (Cascading Style Sheets): seletores, propriedades, cores, fontes, layout com Flexbox e Grid, responsividade e especificidade.

Unidade 5

    Linguagem JavaScript: variáveis, operadores, estruturas de controle, funções, manipulação do DOM, eventos, validações.

Unidade 6

    Frameworks CSS: introdução e uso de ferramentas como Bootstrap e Tailwind CSS no desenvolvimento de interfaces responsivas.

Unidade 7

    Principais bibliotecas JavaScript: introdução ao uso de bibliotecas modernas como React, Vue.js e Angular no contexto de desenvolvimento front-end com foco em UX/UI.

Unidade 1: Design de sistemas interativos

Importância do Foco no Humano: A necessidade de criar designs centrados no usuário.

Variedade de Sistemas Interativos: A diversidade de sistemas com os quais os usuários interagem.

Preocupações de Design: Considerações e desafios no processo de design de sistemas.

Vida Digital: O impacto e a integração da tecnologia no cotidiano.

Habilidades de Designer: As competências necessárias para desenvolver sistemas eficazes.

Unidade 2: UX (User Experience) e UI (User Interface)

Fundamentos: Princípios básicos de UX e UI.

Design Centrado no Usuário: Foco nas necessidades e experiências do usuário durante o desenvolvimento.

Acessibilidade: Criação de designs que atendam a todos os usuários, inclusive os com deficiência.

Heurísticas de Nielsen: Diretrizes para avaliar e melhorar a interface do usuário.

Arquitetura da Informação: Organização e estruturação eficaz do conteúdo.

Responsividade: Design adaptável a diferentes tamanhos e tipos de tela.

Elementos a serem considerados na análise e solução de problemas em UX/UI:

Tecnologias: Ferramentas usadas para resolver o problema

Contextos: Ambiente onde o problema ocorre

Atividades: Ações relacionadas relacionadas ao problema

Pessoas: Usúarios afetados pelo problema

Problema: O foco na análise

Unidade 3: HTML (Hypertext Markup Language)

1. HTML: Linguagem fundamental usada para estruturar conteúdo na web.

2. HTML vs HTML5: Comparação entre versões antigas e modernas do HTML, destacando a evolução dos padrões.

3. Formulários: Elementos de entrada de dados, essenciais para interação com o usuário.

4. Atributos Globais: Atributos que podem ser aplicados a qualquer elemento HTML, aumentando a flexibilidade do código.

5. Portfólio / Currículo: Uso de HTML para criar páginas que mostrem habilidades e experiências profissionais.

6. Plataformas de Publicação: Ferramentas para hospedar e publicar sites criados com HTML.

Unidade 4: CSS (Cascading Style Sheets)

Fundamentos de CSS: Conceitos básicos, seletores e propriedades.

Layout com Flexbox e Grid: Técnicas de layout modernas e avançadas para posicionamento de elementos.

Design Responsivo: Criação de layouts que se adaptam a diferentes tamanhos de tela.

Introdução ao Bootstrap: Primeiros passos com o framework para facilitar o desenvolvimento responsivo.

Containers Bootstrap: Uso eficaz dos containers do Bootstrap para estruturar conteúdos

Modelos Bootstrap: Construção de sites completos usando modelos prontos do Bootstrap.

Unidade 5: Linguagem JavaScript

Variáveis: Declaração e atribuição de valores.

Operadores: Operadores aritméticos e lógicos para manipulação de dados.

Estruturas de Controle: Comandos de decisão e repetição como if, switch, for e while.

Funções: Definição e chamada de blocos reutilizáveis de código.

Manipulação do DOM: Seleção e modificação de elementos HTML dinamicamente.Eventos:

Eventos: Resposta a ações do usuário, como cliques e movimentação do mouse.

Validações: Verificações de dados e formulários antes do envio.

Unidade 6: Frameworks CSS

Introdução ao Bootstrap: Visão geral do framework CSS mais tradicional e amplamente utilizado.

Uso do Bootstrap: Aplicações práticas para criar interfaces responsivas com agilidade.

Introdução ao Tailwind CSS: Apresentação do Tailwind, um framework CSS moderno baseado em classes utilitárias.

Uso do Tailwind CSS: Exemplos práticos de como usar o Tailwind para criar layouts flexíveis e responsivos.

Unidade 7: Principais bibliotecas JavaScript

React: Biblioteca JavaScript usada para construir interfaces de usuário de forma eficiente e modular.

Vue.js: Framework progressivo que permite criar interfaces de maneira flexível e escalável.

Angular: Plataforma completa para desenvolvimento de aplicações web e móveis, com estrutura robusta.

Foco em UX/UI: Ênfase na criação de experiências de usuário agradáveis e interfaces bem projetadas.
